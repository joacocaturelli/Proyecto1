¿Qué es CSS?

Las Hojas de Estilo en Cascada (CSS) son el lenguaje utilizado para describir la presentación de documentos estructurados en HTML o XML. Permite controlar el color, la tipografía, los espacios y los diseños de las páginas web, convirtiendo el esqueleto de HTML en una interfaz atractiva y funcional.

Estructura básica de una regla

Una regla de CSS consta de un selector que apunta al elemento y un bloque de declaraciones encerrado entre llaves.

/* selector { propiedad: valor; } */
p {
  color: blue; /* Declaración */
  font-size: 16px;
}

Selectores esenciales

Los selectores permiten aplicar estilos a elementos específicos de forma precisa.

p { } /* Por etiqueta */
.card { } /* Por clase (más recomendado) */
#header { } /* Por ID (identificador único) */
.nav > a { } /* Hijo directo */
a:hover { } /* Pseudo-clase de estado */
input[type="email"] { } /* Por atributo */

Especificidad (Regla rápida)

Determina qué estilos prevalecen cuando hay conflicto entre varias reglas aplicadas a un mismo elemento.

/* Orden de fuerza (de mayor a menor): */
/* 1. !important (evitar su uso) */
/* 2. #id */
/* 3. .clase, [atributo], :pseudo-clase */
/* 4. etiqueta, ::pseudo-elemento */

Colores y Variables (Custom Properties)

Las variables facilitan el mantenimiento de colores y valores repetitivos en todo el documento.

:root {
  --primary: #3a8de0;
  --text: #1a1a1a;
}

.btn {
  background: var(--primary);
  color: white;
}

Tipografía y texto

CSS ofrece control total sobre las fuentes, tamaños, alineación y espaciado del contenido textual.

h1 {
  font-family: Arial, sans-serif;
  font-size: 2rem;
  line-height: 1.2;
  text-align: center;
}

Modelo de Caja (Box Model)

Define cómo se calcula el espacio de un elemento: contenido, relleno (padding), borde y margen.

.card {
  margin: 16px; /* Espacio exterior */
  padding: 16px; /* Espacio interior */
  border: 1px solid #ddd;
  box-sizing: border-box; /* Incluye padding/border en el ancho */
}

Display: block, inline e inline-block

Define cómo se comporta un elemento en el flujo del documento y cómo ocupa el espacio.

.block { display: block; } /* Ocupa todo el ancho */
.inline { display: inline; } /* No respeta width/height */
.inline-block { display: inline-block; } /* Respeta dimensiones y alinea en línea */

Flexbox: Contenedor e ítems

Modelo de diseño unidimensional para alinear elementos en filas o columnas de forma eficiente.

.container {
  display: flex;
  flex-direction: row; /* u horizontal (row) o vertical (column) */
  justify-content: space-between; /* Alineación eje principal */
  align-items: center; /* Alineación eje cruzado */
  gap: 12px; /* Separación entre hijos */
}

Position (Posicionamiento)

Controla la ubicación de un elemento respecto a su flujo original o su contenedor.

.relative { position: relative; } /* Referencia para hijos absolutos */
.absolute { position: absolute; top: 0; right: 0; }
.fixed { position: fixed; bottom: 10px; } /* Se queda pegado a la pantalla */
.sticky { position: sticky; top: 0; } /* Se pega al hacer scroll */

Backgrounds y Bordes

Permiten definir el estilo visual de la caja del elemento, incluyendo imágenes y colores de fondo.

.hero {
  background-color: #f5f5f5;
  background-image: url('img/hero.jpg');
  border-radius: 12px;
  border-bottom: 2px solid #eee;
}

Transformaciones (Transform)

Modifican el sistema de coordenadas visual del elemento sin afectar el flujo del documento (rotar, escalar, mover).

.caja {
  transform: rotate(45deg);
  transform: scale(1.2);
  transform: translate(10px, 20px);
}

Transiciones básicas

Permiten crear cambios fluidos de estilo entre estados (ej: de normal a hover) en un período de tiempo.

.btn {
  transition-property: transform, background;
  transition-duration: 0.3s;
  transition-timing-function: ease-out;
}

Animaciones con @keyframes

Se basan en definir pasos o "cuadros clave" que el elemento sigue progresivamente a lo largo del tiempo.

@keyframes aparecer {
  from { opacity: 0; transform: translateY(10px); }
  to { opacity: 1; transform: translateY(0); }
}

.modal {
  animation: aparecer 0.5s ease forwards;
}

Responsive Design y Media Queries

Permite adaptar el diseño web para que se vea bien en móviles, tablets y escritorios.

/* Estilos base para móvil (Mobile First) */
body { padding: 10px; }

/* Estilos para pantallas de más de 768px */
@media (min-width: 768px) {
  body { padding: 50px; }
}

Buenas Prácticas de CSS

  • Enlace Externo: Enlaza siempre el CSS externamente con <link> para separar estructura y presentación.
  • Mobile First: Diseña primero para pantallas pequeñas y usa min-width para añadir estilos de escritorio.
  • Clases vs ID: Usa clases (.class) para estilos y evita los ID (#id) para facilitar la reutilización.
  • Box Model: Usa box-sizing: border-box en el reset inicial para un cálculo de tamaños más intuitivo.
  • Animaciones: Usa transform y opacity para animaciones, ya que son gestionadas por la GPU y no fuerzan al navegador a recalcular el layout.
  • Nomenclatura: Asigna nombres claros y significativos a tus reglas de @keyframes y variables.
