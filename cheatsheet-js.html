¿Qué es JavaScript?

JavaScript (JS) es un lenguaje de programación de alto nivel, interpretado y multiparadigma (orientado a objetos, imperativo y funcional). Se utiliza principalmente para dar interactividad a las páginas web, permitiendo actualizar contenido de forma dinámica y controlar elementos multimedia.

Variables y Alcance (Scope)

Las variables son contenedores para almacenar valores. JavaScript utiliza tres palabras clave principales para declararlas:

  • const: Para valores que no cambian (no reasignables). Tiene alcance de bloque].
  • let: Para valores que pueden cambiar. Tiene alcance de bloque.
  • var: Forma antigua de declarar variables; se recomienda evitarla por su alcance de función poco intuitivo.
const PI = 3.14; // No puede cambiar
let contador = 0; // Puede cambiar
var nombre = "Juan"; // Evitar usar var

Tipos de Datos Básicos

JavaScript es un lenguaje de tipado dinámico, lo que significa que las variables pueden cambiar de tipo en tiempo de ejecución.

const texto = "Hola"; // String
const numero = 42; // Number
const esReal = true; // Boolean
const lista =; // Array (Objeto especial)
const usuario = { id: 1 }; // Object
let x; // Undefined (declarada sin valor)
const y = null; // Null (ausencia intencional)

Operadores esenciales

Son fundamentales para la lógica de los programas y la toma de decisiones.

// Comparación
5 == "5"; // true (comparación débil, convierte tipos)
5 === "5"; // false (comparación estricta, sin conversión)

// Lógicos
true && false; // false (AND)
true || false; // true (OR)
!true; // false (NOT)

Funciones

Las funciones son bloques de código diseñados para realizar tareas específicas. Las Arrow Functions (ES6) ofrecen una sintaxis más concisa.

// Declaración clásica
function sumar(a, b) {
  return a + b;
}

// Arrow function (Flecha)
const multiplicar = (a, b) => a * b; // Return implícito en una línea

Arrays y Métodos de Transformación

Los arrays son colecciones ordenadas de datos. Los métodos modernos permiten manipularlos de forma declarativa .

const nums = [4-6, 15];

// map: Transforma y crea un nuevo array del mismo tamaño
const dobles = nums.map(n => n * 2); //

// filter: Crea un nuevo array con elementos que cumplen una condición
const pares = nums.filter(n => n % 2 === 0); //

// find: Devuelve el primer elemento que cumpla la condición
const encontrado = nums.find(n => n > 2); // 3

Objetos

Colecciones de propiedades representadas en pares clave-valor.

const user = { name: "Ana", age: 30 };

// Métodos útiles de Object
Object.keys(user); // ["name", "age"] [22, 23]
Object.values(user); // ["Ana", 30] [22, 24]
Object.entries(user); // [["name", "Ana"], ["age", 30]]

Sintaxis Moderna (ES6+)

Facilitan el desempaquetado de datos y la copia de estructuras.

// Desestructuración de objetos
const { name, age } = user; // Extrae propiedades a variables

// Spread operator para arrays
const arr1 = [1, 2];
const arr2 = [...arr1, 3, 4]; // Combina y añade elementos

// Spread operator para objetos
const user2 = { ...user, city: "Madrid" }; // Copia y añade propiedad

Selección de elementos en el DOM

El DOM representa la página como un árbol de nodos que JS puede cambiar.

const titulo = document.getElementById('main-title'); // Por ID
const parrafo = document.querySelector('.texto'); // Primer coincidencia
const items = document.querySelectorAll('li'); // Todos (NodeList)

Modificación de Contenido y Estructura

Permite alterar lo que el usuario ve y la estructura de las etiquetas.

const caja = document.querySelector('.box');
caja.textContent = "Nuevo texto seguro"; // Solo texto
caja.innerHTML = "<strong>Importante</strong>"; // Inyecta HTML

// Crear e insertar elementos
const nuevoLink = document.createElement('a');
document.body.append(nuevoLink); // Añade al final

Eventos (Interactividad)

Permiten responder a acciones del usuario como clics o envíos de formularios.

const btn = document.querySelector('#btn');
btn.addEventListener('click', (e) => {
  console.log("¡Botón pulsado!");
});

// Prevenir comportamiento por defecto (Formularios)
form.addEventListener('submit', (e) => {
  e.preventDefault(); // Evita recargar página
});

LocalStorage (Persistencia)

Guarda datos de forma permanente en el navegador del usuario.

// Guardar (solo strings)
localStorage.setItem('user', JSON.stringify({id: 1}));

// Recuperar y convertir de vuelta a objeto
const data = JSON.parse(localStorage.getItem('user'));

Buenas Prácticas de JavaScript

  • Declaración: Usa const por defecto y solo let si el valor va a cambiar. Evita var.
  • Nomenclatura: Utiliza nombres descriptivos en camelCase (ej: obtenerDatosUsuario).
  • Comparación: Emplea siempre el comparador estricto (=== y !==) para evitar errores de tipo.
  • Responsabilidad Única: Mantén las funciones cortas y enfocadas en una sola tarea.
  • Seguridad del DOM: Prefiere textContent sobre innerHTML para evitar ataques de inyección de código (XSS).
  • Ubicación de Scripts: Coloca las etiquetas <script> al final del <body> para mejorar la velocidad de carga inicial.
